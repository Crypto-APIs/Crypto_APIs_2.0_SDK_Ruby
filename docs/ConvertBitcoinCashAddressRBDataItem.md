# CryptoApis::ConvertBitcoinCashAddressRBDataItem

## Properties

| Name | Type | Description | Notes |
| ---- | ---- | ----------- | ----- |

require 'crypto_apis'

instance = CryptoApis::ConvertBitcoinCashAddressRBDataItem.new(
